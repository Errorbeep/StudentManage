# StudentManage
a student info management system

### 技术栈
+ Java/JavaFX
+ Oracle Database
+ CSS/FXML

### 运行环境
+ Java 8 或者其它带有JavaFX的java版本，如librica-java full
+ Oracle 11g

### 主要功能
+ 学生信息增删改查
+ 学生成绩增删改查/统计分析
+ 管理员登录/注销/修改
